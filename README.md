"# Console-Based-Online-Shopping-System"
A simple console-based Online Shopping Application developed using Java and OOP concepts.
This project supports both Seller and Buyer roles with secure Login & Registration.

🚀 Features
## User Authentication
Register as Seller or Buyer
Login using ID & PIN
Role-based access control

 ## Seller Functionalities
Add Product
Update Product
Delete Product
View All Products

## Buyer Functionalities
View All Products
Add Products to Cart
View Cart
Generate Bill & Buy Products

## Concepts Used
Object Oriented Programming (OOPS)
Classes & Objects
ArrayList
Iterators
Role-based Authentication
Console Menu Driven System
